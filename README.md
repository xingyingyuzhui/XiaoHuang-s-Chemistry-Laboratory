# 小黄的化学实验室

面向**中学化学教学与自学**的本地应用：在教室大屏、教师机或学生自习时，用统一界面完成周期表讲解、3D 分子与反应示意、计算练习、电子排布演示和课堂互动。

数据全部保存在本机 SQLite；可选接入 **DeepSeek API** 做 AI 出题、提示、分子/反应生成等（不上传本地名单与试卷以外的业务数据到第三方，仅按调用转发提示词）。

**当前版本：v2.0.0**

## 功能详解

### 1. 元素周期表

适合课堂大屏讲解与学生对照查阅。

- **标准布局**：18 族 × 7 周期主表，含镧系 / 锕系展开行
- **分区配色**：s / p / d / ds / f 区颜色与图例同源，并随主题联动
- **显示开关**
  - **分区**：开/关分区底色
  - **分界**：金属与非金属阶梯分界线
- **侧栏详情**（点选元素后）
  - 元素符号、名称、序号
  - 电子排布与相关课堂提示
- **侧栏标注**：电子层 / 0 族电子数等辅助信息列

---

### 2. 3D 分子

用 Three.js 球棍模型展示中学常见分子结构，并串联「键 → 性质 → 反应」讲解路径。

#### 2.1 分子列表与模型

- **内置分子**（示例）：H₂、O₂、N₂、Cl₂、O₃、H₂O、H₂O₂、HCl、H₂S、NH₃、CO、CO₂、SO₂、SO₃、NO、NO₂、NaCl、CH₄、C₂H₆、C₂H₄、C₂H₂、甲醇、乙醇、甲醛、乙酸、苯、葡萄糖等常见物质
- **编辑模式**：排序、删除自定义项等（内置项策略以界面为准）
- **3D 操控**
  - 左转 / 右转 / 上仰 / 下俯
  - 放大 / 缩小
  - **慢转**（自动旋转）、**复位**视角
  - **标签**：显示或隐藏原子元素符号
- **信息卡片**
  - 名称、简介
  - **课标向物质卡片**：类别、用途、性质要点、安全注意（与分子 id 对应）
- **化学键点选**：在 3D 中点击一根键，弹出键型与中学表述说明

#### 2.2 AI 生成分子

- 入口：左侧「＋」
- 输入名称或化学式，生成示意用球棍模型并保存到本地库
- **适用**：高中常见小分子（建议原子数约 20 以内，如乙醇、苯、乙酸、葡萄糖）
- **拦截**：紫杉醇、阿莫西林等复杂药物 / 过大结构会拒绝生成，避免残缺模型误导课堂

#### 2.3 常见反应（剧院式播放）

右上角 **「反应」** 打开当前分子关联的反应列表。

**内置反应示例（示意动画，非严格机理模拟）**

| 反应 | 方程式（示意） |
|------|----------------|
| 乙烯与溴加成 | C₂H₄ + Br₂ → C₂H₄Br₂ |
| 乙烯加聚 | n C₂H₄ → ‑[CH₂‑CH₂]ₙ‑ |
| 乙醇催化氧化 | 2 C₂H₅OH + O₂ → 2 CH₃CHO + 2 H₂O |
| 乙酸乙酯化 | CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O |
| 乙炔与溴加成 | C₂H₂ + 2 Br₂ → C₂H₂Br₄ |
| 苯的溴代（示意） | C₆H₆ + Br₂ → C₆H₅Br + HBr |
| 甲烷氯代（第一步） | CH₄ + Cl₂ → CH₃Cl + HCl |
| 氢气燃烧 | 2 H₂ + O₂ → 2 H₂O |
| 乙烯水化制乙醇 | C₂H₄ + H₂O → C₂H₅OH |
| 氢气与氯气 | H₂ + Cl₂ → 2 HCl |

**播放体验**

- 底部 **剧院 dock**：步骤说明、方程式、讲解提示
- **暂停 / 重播 / 关闭**
- **进度条可拖动**回顾
- **变化历程**列表：点击某步跳转回顾
- 3D 场景随步骤切换反应物 / 过渡 / 产物焦点

**AI 添加反应**

- 描述一条高中常见反应 → 生成预览（可选 4 / 5 / 6 示意步）→ **人工确认后再保存**
- 建议课前核对方程式与步骤表述

**导入导出**

- **导出反应包** / **导入反应包**（JSON），便于备课机之间迁移

### 3. 计算

左侧二级导航，三项工具：

| 子功能 | 说明 | 是否需要 API |
|--------|------|----------------|
| **摩尔质量** | 输入化学式，计算相对分子质量；分元素质量贡献明细；内置示例快捷填入 | 否 |
| **配平方程** | **本地配平** + 原子守恒校验；可选 **AI 建议** 辅助疑难式 | 本地否 / AI 是 |
| **分步计量** | 粘贴文字题（如「2.5 g 氢气完全燃烧需要多少克氧气？」），AI 分步化学计量解答 | 是 |

适合：课堂随手算、作业校对、板书前快速验证配平结果。

---

### 4. 电子排布

- **覆盖范围**：1～36 号元素
- **3D 示意**：玻尔多壳层模型（教学示意，非量子轨道精确图）
- **侧栏列表**：点选切换元素；显示电子排布式等信息
- **编辑排序**：列表可拖拽排序并持久化（方便按授课顺序排课）
- **视角控制**：与 3D 分子相同的旋转 / 缩放 / 慢转 / 复位

---

### 5. 课堂

左侧二级导航四个区块：

#### 5.1 智能出题

- **年级范围**、**章节 / 主题**（可多选）、**难度**、**题量**（1～10）
- **对错显示**策略可配置
- 一键 **生成题目**（DeepSeek）
- 答题过程：
  - 交卷 → 本场得分与逐题回顾
  - **导出 Markdown**（便于打印或发到学习群）
  - **AI 分析总结**（本场薄弱点）
  - **AI 提示 / AI 解答**（单题助教，有次数限制）
- **历史做题总结**：近期场次正确率等统计
- 试卷内容服务端快照存储，批改与回顾更稳定

#### 5.2 错题本

- 做错自动收录；做对后可移出（以界面逻辑为准）
- 列表复习；支持刷新
- 可对错题继续使用 AI 提示 / 解答（计入助教限流）

#### 5.3 随机点名

适合课堂提问暖场或抽查。

- **班级名单**：单条添加、删除
- **批量导入**：一行一个姓名，或 CSV 姓名列
  - **追加导入** / **清空后导入**
- **卡牌轮转**动画后定格姓名
- **再来一次**连续抽点
- 名单存本地数据库（`/api/students`）

#### 5.4 实验探究

三级结构：课堂 → 实验探究 → 具体实验脚本。

内置脚本示例：

| 实验 | 类型 | 要点 |
|------|------|------|
| 实验室制氧气（高锰酸钾） | 气体制备 | 装置、棉花、防倒吸、验满 |
| 氢气燃烧与验纯 | 性质实验 | 验纯、淡蓝火焰、键变化 |
| 实验室制二氧化碳 | 气体制备 | 盐酸 + 大理石、向上排空气 |
| 酸碱中和 | 性质实验 | 中和过程与现象 |
| 乙酸乙酯化 | 有机制备 | 酯化条件与注意 |

每个脚本含：标题、类型、化学方程式、**安全提示**、分步步骤（含**化学键变化**中学表述）、现象摘要。便于实验课前口述或投影。

## 设置与品牌

右上角齿轮打开设置抽屉：

| 区块 | 内容 |
|------|------|
| **标识** | 自定义头像（PNG / JPG / WebP / GIF，建议 ≤ 500KB）、标题文字；可恢复默认 |
| **主题** | 见上表 |
| **默认页** | 启动进入：周期表 / 3D 分子 / 计算 / 电子排布 / 课堂 |
| **AI** | API Base URL、API Key、模型（如 `deepseek-v4-flash` / `deepseek-v4-pro`） |

设置写入本地数据库；AI 相关配置在服务端做深合并与校验，`apiBase` 走白名单，降低误配与注入风险。

## AI 与限流

需在设置中填写有效 DeepSeek API Key 后，下列功能才可调用云端：

- AI 生成分子 / AI 添加反应  
- 智能出题、单题提示与解答、本场 AI 总结  
- 配平 AI 建议、分步计量  
- 课间小知识 AI 生成  

**默认限流（约每小时，偏保守，防止课堂连点刷爆账单）**

| 类别 | 上限（约） |
|------|------------|
| 课间小知识（tip） | **40** 次 / 小时 |
| 助教（提示 + 解答等） | **20** 次 / 小时 / 类 |
| 全局 AI 调用 | **120** 次 / 小时 |

超出后接口返回明确提示，稍后自动恢复。本机教学一般不做登录鉴权。

**分子安全校验（摘要）**：原子数上限、键几何合理性、复杂药物黑名单等；课堂场景以「能讲清楚的小分子」为准。

## 开发运行

需要 **Node.js 18+**。

```bash
npm install
npm --prefix server install

# 终端 1：后端
cd server && npm start

# 终端 2：前端
npm run dev
```

浏览器打开：http://localhost:5173/  
（Vite 将 `/api` 代理到后端默认端口 3000）

### 打包

```bash
# Windows Electron 安装包（在 mac 上通常可交叉打包）
npm run dist:win

# macOS Apple 芯片 dmg
npm run dist:mac

# 路线 B：Windows 轻量单文件 exe（pkg）
npm run build:exe
```

产物目录：

- `dist-electron/` — Electron 安装包 / dmg  
- `dist-exe/` — 便携 exe  

上述目录不入库。

## 技术架构

| 层 | 技术 |
|----|------|
| 前端 | Vite、原生 ES Module、Three.js |
| 后端 | Express |
| 数据库 | sql.js（SQLite 文件，落盘本地） |
| 桌面 | Electron（内嵌 Express + 静态前端） |
| AI | DeepSeek Chat Completions（服务端代理） |

**主要 API**

| 路径 | 用途 |
|------|------|
| `/api/molecules` | 分子 CRUD / 列表 |
| `/api/settings` | 品牌、主题、默认页、AI 配置 |
| `/api/ai` | 分子生成、小知识、出题 / 提示 / 解答、反应 / 配平 / 计量等 |
| `/api/quiz` | 练习场次、错题本、出题快照 |
| `/api/reactions` | 化学反应列表与导入导出 |
| `/api/students` | 课堂点名名单 |
